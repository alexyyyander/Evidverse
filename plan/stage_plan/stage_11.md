# Stage 11: 前端基础架构

## 目标
搭建 Next.js 前端项目，配置基础样式和状态管理。

## 功能列表
1. **脚手架**: Next.js + TypeScript + Tailwind CSS。
2. **组件库**: 集成 shadcn/ui 或其他 UI 库。
3. **状态管理**: 配置 Zustand Store。
4. **API Client**: 封装 Axios/Fetch 请求后端。
